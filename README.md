# Personal Expense Tracker (Python)

## Overview
Track daily expenses and visualize spending by category using Python.

## Features
- Add daily expenses (date, category, amount)
- View all expenses
- Visualize total spending per category
- Track expenses over time

## Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## How to Run
1. Open notebook in Google Colab
2. Run all cells
